Nem-koaas a ye kibare.
Nem-koaas a ye n wa n da naafo. La, a yãka yam, n yeel t’a ka na n kʋ naafã zõaag sẽn be zĩig ning ye. La a yikame, n na n kẽn n kẽ we-raoog pʋgẽ ne naafã, n na n bao zĩig ning zõaag sẽn ka be, n kʋ-a.
La, a bi-bilf yikame, t’a na n tũ-a lame, la, a ba wã tõdgame. La biigã baoo tompeglem n ning a ba wã korgã pʋgẽ, n yaool n poog võ-bil korgã tẽngre, t’a ba wã sã n na n looge, bɩ a tõog n tũ tompeglmã vũkã, n pʋg-a.
Rao wã sɩd wa n yikame, n dɩk a korgã n bobe, n pid a naafã wĩiri, n tat n kẽed weoogã. La, a biigã yikame, n get tompeglmã, n pʋgd a ba wã. A ba wã wa n wẽnegame, n yã biigã t’a pʋgd-a. A ka le tõe n yeel biigã t'a lebg n kuil ye. Bala, b zoe n bee weoogã sʋka. La, b naaga taab n tar kẽna marsã.
La rao wã yeela a biigã t’a sã bĩndu. La biigã sãame tɩ zõosã wa. Tɩ rao wã yet tɩ naafã kʋʋb zĩig pa ka ye. Tɩ b le tar kẽn n tar kẽna. A le wa n yeela a biigã t’a sã, tɩ biigã sɩd sã, la zõosã yɩɩ wãa… n wa. La rao wã yeelame, tɩ naafã kʋʋb zĩig nan ka ta ye. Tɩ b sɩd be taoor sãagre. A le wa n yeela a biigã t’a sã, tɩ biigã sã, tɩ zõos ka wa ye, t’a yeele, tɩ marsã, naafã kʋʋb zĩig taame.
B loee naafã n lub n kodg-a ne sʋʋga. Rao wã yaool n tʋma biigã, t’a kẽng n tɩ mok bugum, n wa tɩ b sẽ naafã sãoore. Biigã kẽngame n na n mok bugmã, n tɩ mik tɩ bugmã soab yaa kinkirga, la a sokame : «Ãnda n wa n kogsd m yẽdga ?». Tɩ biigã leoke : «Yaa m ba n tʋm maam, tɩ m wa mok bugum n wa t’a sẽ sãoore ” Tɩ kinkirgã sok-a : «M sã n wa, m na n paamame bɩ ?». La biigã leok-a lame : «N-yẽe !» Kinkirgã sẽn kẽng n tɩ ta b zĩigẽ wã, a yeel-b lame : «Rɩk-y pʋ-teedã n wa ne». Tɩ b rɩk n kõ-a t’a reeg n vole. N le yeel-ba : «Wa-y ne gɛyã ne bãgdã». Tɩ b kõ-a t’a reeg n vole. N le yeel-b yasa : «M data zugã ne gãongã n paase». Tɩ b rɩk n kõ-a, la b na n zoe. B zoe n tɩ kẽe roog a ye pʋg n na n solge. Mik, roogã yaa kinkirgã roogo. A sẽn wa n ta, a maaga a meng n wʋb nemdã fãa laag pʋgẽ, n yaool n dɩk a sag-rʋk n na n moe sagbo. A sẽn wa n moe n sa, a rɩka laagã ne rɩɩbã, n bĩng a poorẽ wã.
La biigã ne a ba wã sẽn da solg pĩirã poorẽ wã, rɩka rɩɩbã n dɩ tɩ yɩ-b noogo, hal tɩ zẽedã wa sa. Nemdã yĩnga, zẽedã sɩd yɩɩ noog tɩ looge. Biigã yikame n na n bõos zẽedã. T’a ba wã yɩ tao-tao n yeel-a, t’a ra mak, tɩ kinkirgã na n kʋ-b lame. Tɩ biigã zĩẽ n zĩẽ, n wa kelme, n yeele : «Pʋg-yãanga, m kota zẽedo!» La kinkirgã yikame, n wudg n tɩ tigim we-rũmsã fãa n yeel tɩ : «pʋg-yãang-m-kota-zẽed n be m doogẽ wã, n dat n kʋ maam».
We-rũmsã fãa wa n naaga taab n wa gũbg roogã, n na n gese. B yaool n tʋma m ba Yõngr t’a tɩ ges bõe n be roogã pʋgẽ. La m ba Yõngr sẽn wa n tar lamsg n dabd n na n gese, la biigã sẽn yã-a, n wẽ-a n kʋ.
Tɩ b yeel m ba Yũug t’a kolg n gese. T’a kolg n yaees n lebge. Tɩ b le yeele, tɩ m ba Rãaneg kolg marsã, n ges tɩ yẽ yubl n woagem la!
T’a sɩd yɩ gond-gond n kolge, tɩ biigã fiuuga a yublã ne sʋʋga. Tɩ m ba Rãaneg tar po-tulli, n tɩ rok n lʋɩ zʋko! Lebga yelle.
Tɩ m ba Yʋgempend yet tɩ bõe la wã? Tɩ m ba Wobg rãmb zoe bilf-bilfu, tɩ bãmb ra wa lʋɩ b nao-bogd pʋsẽ, n pa tõog n yi ye.
Rẽ wã, ned fãa sã n na n saag n mak, n maan bũmbu, bɩ a mag wʋsg la a kos saglgo, sã n pa rẽ, a sẽn na n maanã, wat n tõe n wa nafa ned a to n bas-a.
