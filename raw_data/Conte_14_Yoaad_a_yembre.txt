Yoaad a yembr kibare.
Rao a ye, n wa n be tẽng a yembr pʋgẽ. La, rao wã ra nonga pagb wʋsgo. Ned bi-neere, bɩ a pʋg-neere, ka pɩʋʋgd a taoor ye. Hal tɩ saka neba, tẽngã la tẽn-yagsã fãa, wa gomd a zĩnd-n-ta-kõb-wẽndlmã yelle.
La, sẽn wa n beẽ n beẽ, kinkirg n lebg pʋg-neere, n wa rɩk-a n maan a sɩda, tɩ b be ne taaba. La, bʋdb wakat sẽn wa n ta, rao wã sã n kẽng ne a pagã, n tɩ bʋd ki wã, n lebg zaabr n kuili, pagã yikda tõd-beoog pĩnda, n tɩ bool a koden-n-taasã, tɩ b wa gaas sẽbdã n dɩ ki wã. Rao wã sã n yik yibeoogo, n tɩ ta a pʋʋgẽ wã, a sũurã sãamda hal wʋsgo, t’a le sɩng bʋdbo. A sã n roog zaabr n kuili, vẽkembeoogo, a sã n leb n ta pʋʋgẽ wã, a mikdame, tɩ bʋd-bogdã fãa le gaasame. Talla woto, tɩ rao wã, ka paam ki yʋʋm-kãng ye. La, yʋʋmd ning sẽn pʋgdã, b le kẽng n tɩ bʋdame, la b lebg n kuili. Vẽkembeoogo, rao wã sẽn le kẽng pʋʋgẽ wã, a yãame tɩ kodensã le wa n gaasa bʋd-bogdã fãa tẽma, n dɩ ki wã. La, tãoos a yembre, n wa n na n pʋge, n gũus n gese. Tõd-beoog sẽn wa n ta, kodensã le tũu taab wʋsgo, n wa gaasd ki wã. La, tãoosã maaga a menga, n lall n kelgd-b goamã. La, pʋʋgã soab pag yeela a taabã : «Rũnda, d rɩ, la d kell m sɩd m ba Tẽedba.»
Tãoosã sẽn wʋm dẽ, a loogame, n tɩ togs pagã sɩd kibarã. Daar a yembre, rao wã ne a pagã wa n bee pʋʋgẽ wã n bʋtẽ, la b tar sõasga. Rao wã yeelame : «Wa tɩ d rɩ, la d kell m ba sɩd a Tẽedba.» T’a pagã yeel-a, t’a ra le saag n gom woto ye, tɩ yaa yẽ kisgu. B sẽn ta yiri, wa tar sõasg yasa, rao wã le yeelame : «Wa tɩ d rɩ, la d kell m sɩda m ba Tẽedba.» Pagã bãngame, tɩ rao wã wʋma bãmb sõasgã. La, a wa n menmame, tɩ rao wã ka bãnge, a sẽn teesẽ n ye. Rẽ tɛka, m ba Tẽedb basa pagb roobo.
Rẽ wã yĩnga, zʋgd buud fãa, tara a rolbo.
