Kũum ne dasãare !
Rao n wa n kẽng bagre. La, bagrã soab yeel-a lame, t’a bao yɩbrg-vɩɩga, n bila a yĩngã, la a rɩke, n maan maoongo, rasem a yopoe pʋgẽ, sã n ka rẽ, a na n kiime. Rao wã pelẽnga, yɩkame, t’a yik n dɩk weoogã sore, n tar yãbr n teesdẽ. La, a wa n tɩ sega pʋg-yãanga a yembre. La, a sok-a lame t’a yeele : «Mam kẽngame, n na n tɩ bao m yɛl võore, bagr soab nengẽ, t’a yeele, tɩ m bao yɩbrg-vɩɩg n bil m yĩngã, n maan maoongo. La, m na n paam-a la yɛ?». La pʋg-yãangã yeelame : «Tɩ sã n ya woto tɛka, ka toog ye. Bao dasãare, n kẽng zĩ-peelẽ wã, n tɩ gãand zãmbeelle, f poorã zugu, n nodg f ninã, wa f kiime. Yɩbrdã sã n yã foo, b na n tẽeme, tɩ yaa kũum, tɩ b na n dɩ. Tɩ f paam-ba, n yõke.»
Rao wã kẽngame, n sɩd maane, wa pʋg-yãangã sẽn yeelã. Sẽn yɩ bilf bala, la yɩbrg naab sẽn laale, n wa ta n sãage, n kẽng n tɩ ta, n zomb tɩɩga, n yetẽ : «Õ’õ, mam nan pa rɩ kũum, ne dasãar ye». Rẽ poore, yʋʋm a tãab yɩbrgo, n wa n ta, n laandẽ, n dat n sigi, n dɩ-a. La, yɩbrg naab bool-a lame, n soke : «Fo sẽn dog tɛka, fo yãa kũum ne dasãare ?» T’a yet tɩ ayoo ! «Bɩ, f loog n bao zĩiga, n zaoonde». Sẽn kaoos bilfu, la yʋʋm a yiib yɩbrg sẽn wa n ta n laandẽ, n dat n dɩ rao wã, tɩ yɩbrg naab bool-a n soke : «Fo sẽn dog tɛka, fo rɩɩ kũum ne dasãare? » T’a yet tɩ ayoo! «Bɩ, f bao zĩig n zaoonde».
Yʋʋmd a yembr yɩbrg me, n ta, n laandẽ. Tɩ b naabã sok-a : «Fo sẽn dog tɛka, fo rɩɩ kũum ne dasãare ?» La, a leokame, tɩ yẽ sẽn dɩ a buudu, yaa wʋsgo. La, yɩbr naab yeel-a lame, t’a gũus a menga, la a ka kelg ye. A yɩkame, n tɩ zomb rao wã zugu, n na n dɩ-a, tɩ rao wã be a nao wã gãdbẽ, n tar a mengã yĩng bilbu.
 
Rẽ wã, nin-kẽem sã n sagl biiga, bɩ a reege, la a ra paoog-a, t’a yãa pĩnda.
