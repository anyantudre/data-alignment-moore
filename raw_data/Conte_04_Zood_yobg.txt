Zood yõbg yelle.
Wakat fãa, kinkirg a yembr n yit weoogã pʋgẽ, n lebgd ned n wat n dood tẽn-bil a ye raaga. La, a wa n yõga zood ne rao a ye. La, a sã n wa raag fãa, b bee ne taaba.
La, rao wã n yikame, n zomb a wed-moaag n na n tɩ kaag-a. La, a sẽn wa n ta zakẽ wã n pʋʋse, a zoa wã sẽn ya kinkirgã, sɩd yiime n pʋʋs-a, n lebg n kẽ a roogẽ. La, vugr bala, zakã lebga kãgre. Tɩ rao wã ket n zĩẽ, sẽn yi bilfu, la kinkirgã ba wã la a ma wã, tɩ yi n pʋʋs-a n lebg n kẽ. La, rao wã wed-moaagã wa n kʋmame, t’a tik n yiki, n tɩ kolg-a. La wed-moaagã yeel-a lame : «Mam yãbr a tãab-n-soabã sã n mik tõnd ka, b na n kʋʋ tõndo». La rao wã lebgame, n tɩ zĩndi. La wed-moaagã le kʋmame, tɩ rao wã yik n wudg n tɩ ta, n yɩk n zomb wed-moaagã, n na n zoe n kuili. La, a zoa wã me lebga neda, n zomb a wed-moaaga, n na n dig-a. B sẽn tar taab rigbu, rao wã wed-moaagã wa n yaame, tɩ bɛ wã ket n pʋgd-a. Tɩ rao wã yeel a wed-moaagã : «Sã n yaa m ba ne m ma n da foo n kõ maam, bɩ f ra sak tɩ yãnd paam maam ye». Wakat-kãnga, rao wã kẽem bãnga a yamẽ, t’a yao wã bee yell pʋga. A tik n yikame, n dɩk a bug-raoogo, n bob n yɩk n zomb a wed-moaagã, n dɩk sorã, n na n babs-ba. La, a wa n tool n ne-b, b sẽn wate, tɩ wubsgã tʋke. T’a yalse, n maneg a bug-raoogã, n tees n gũudẽ. La, a yao wã wa n pɩʋʋgame, t’a kẽemã kell n tees bug-raoogã, ned ning sẽn pʋgdã zugu, n wẽ, tɩ kinkirgã yak n lʋɩ n ki.
Rẽ wã, zood yõgda ne gũusgu.
