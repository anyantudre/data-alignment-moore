Pag a yembr kibare.
Pag a ye, n wa n be tẽng a yembr pʋgẽ. La, a ra yaa sũ-kĩ-neda, la zoe-bõn soaba, n le yaa wagdre. A pʋg-taabã la yagsã teed sã n menme, sã n tar-n-tarẽ, yaa pagã nengẽ, la b yãt teedã. Hal, tɩ sakã neb fãa, wa gomd a yelle.
Kãang a yembr me, n be tẽngã sʋka, t’a raadã pa wɩɩsd ye. La, pagã wa n yika daar a yembre, n tɩ wɩɩs kãangã raado. Kãangã zĩnã yɩkame, n zõd pagã, n yet t’a rũng-a poor n kuili. Tɩ pagã sɩd make, n kong n buk-a, n ta yiri, n yalse, n yeel-a : «Sig tɩ m lub m raado, t’a tõdge, sig tɩ m moe sagbo, t’a tõdge, tɩ pagã rũng-a n tar-a n gilimd ne zakã, la a tar yãbre, tɩ sakã nebã fãa, wa gũbg-a, n getẽ.
La, zĩnã basame, t’a nams sõma, t’a yaool n wa yeel-a : «Rɩk raadã n tɩ lebs kãangã pʋgẽ tao-tao, sã n ka rẽ, mam ka na n bas-f ye». La, pagã sɩd talla raadã n tɩ lebs kãangã pʋgẽ, n na n paam a menga.
Rẽ wã yĩnga, sẽn kisa, sɩd kisame. Sẽn ka maande, sɩd ka maand ye.
