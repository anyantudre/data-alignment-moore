M ba Soaamb la m ba Katre kibare.
Daar a ye, noosã maana tigr n na n sao. La, sẽn na n yɩlẽ, tɩ reemã yɩ noogo, m ba Soaamb kẽnga ne kʋnde, koom ne guugu, n tɩ ta n pʋʋs-ba. La noosã soka m ba Soaamba : «Yaa noog ne kʋnde, koom la guugu ? ». La, m ba Soaamb leok-b lame : «Kʋndã, yaa sẽn na n tar n wẽedẽ, tɩ y saoodẽ. Guugã, yaa sẽn na n tar n gũud yãmba. Koomã, yaa sẽn na n tar n walgd zĩigã, tɩ wubsgã ra yikdẽ ye.» La, m ba Soaamb wa n sɩnga kʋndã wẽebo, hal tɩ noosã tar saooga. M ba Soaamb wa n basa kʋndã wẽebo, la a yɩ tao-tao, n dʋ tɩɩgã ne koomã, n walge. Noosã zoe n kẽe roogẽ, n tẽed tɩ yaa saaga n niidẽ. T’a sig n dɩk a guugã, n kẽ n pãb noosã, n kʋ, n wʋk n sui a waoogẽ, n kuil n tɩ kõ a paga, t’a maneg n dʋg tɩ b wãbe.
M ba Katr pagã, wa n na n moka bugum, m ba Soaamb zakẽ wã. La, b yãka nemdã n kõ-a, t’a kuil n tɩ wãb ne a sɩda. La, m ba a Katr yeela a pagã : «Ad nemdã yaa noog dɛ». La, a pagã sũur puugame t’a yeel-a  : «Fo ka bark ye. Fo taaba fãa wat n kõta b pagbã rɩɩbo, tɩ b rɩtẽ. La fo zɩ, n wa ne rɩɩb ka ye».
M ba Katr sãdame n tɩ nek m ba Soaamb n soke : «Fo maanda wãn to-to, n paamd nemdã woto fãa, n wat n dɩt ne f zak-rãmba? » La, m ba Soaamb yeel-a lame : «Beoogo, bɩ y wa ne kʋnde, koom la gũug tɩ d kẽnge». Katr tɩ maka gõeem, n kong n yik n yõgneg bugum n wa yeel m ba Soaamba, tɩ beoogã vẽegame, la, a yik tɩ b looge. Tɩ m ba Soaamb yeel-a : «Kẽng-y, tɩ pʋg-yãang sã n tĩs bɩ y wa tɩ d looge». M ba Katr tɩ baoo tɩpare, n to, n kẽng n tɩ ning pʋg-yãang kokorẽ. La, pʋg-yãang tĩsame t’a lebg n wa yã m ba Soaamba, n yet t’a yik tɩ b looge. T’a Soaamb yeel-a : «Kẽng-y tɩ no-raoog sã n kũmi, bɩ y lebg n wa». M ba Katr kẽngame, n tɩ wẽoog no-raoog kokor tɩ noaagã kelme, t’a lebg n wa. M ba Soaamb yaool n mak n kongame, n yiki, tɩ b tũ taab n looge. La, a Soaamb yeela m ba Katre : «Y sã n ta, bɩ y pʋʋs-b fãa. B sã n sok teed nins yãmb sẽn zãadã yelle, bɩ y yeel-ba : «Kʋndã, yaa sẽn na n tar n wẽedẽ, tɩ y saoodẽ. Koomã, yaa sẽn na n tar n walgd zĩigã, tɩ wubsgã ra yikdẽ ye. Guugã, yaa sẽn na n tall n kẽ n pãb yãmb n kʋ».
B sẽn wa n ta, m ba Katr pʋʋsa noosã, tɩ b sok-a : «M ba Katre ! kʋndã, koomã, la guugã, yaa sẽn na n tar n maand bõe ?». Tɩ m ba Katr yeele: «Koomã, yaa sẽn na n tar n walgd zĩigã, tɩ wubsgã ra yikdẽ ye. Kʋndã, yaa sẽn na n tar n wẽedẽ, tɩ y saoodẽ. Guugã, yaa sẽn na n tall n kẽ n pãb yãmb n kʋ».
M ba Katr wa n sɩd sɩnga kʋndã wẽebo, tɩ noosã me sɩng saooga, wa yell ka be ye. La, m ba Katr wa n yikame, n dɩk koomã, n walge. Tɩ noosã zoe n kẽ roogã pʋga, n yɩ tao-tao, n pak fenetrã, n bas vɩʋʋgo. A Katr me, yɩɩ tao-tao, n wa ta, n na n pãb-b ne guugã. Tɩ noosã tũ fenetrã, n yi, tɩ b kõbdã sʋʋg n pa. M ba Katr sẽn yã rẽ wã, a tẽedame, tɩ yaa noosã, t’a kẽng n ta, n kool n sui a waoogẽ, n tall n kuili. A sẽn ta a yiri, a yeela a pagã : «Vaag baag bɛndã, n lob tãmpʋʋrẽ, tɩ rɩ-bɛd waame». La, a pagã sɩd vaaga baag bɛndã, n lobe. La, m ba Katr sẽn zuk waoogã, n pẽes no-kõbdã, n tẽed t’a na n yãa noaag a pʋgẽ, a pa yã baa fɩ. T’a sũurã sãame, t’a yãs sɩɩ...
Rẽ wã, pẽdgr yiisda yam, n tõe kɩte, t’a soab kẽ moogo, t’a baasg lebg sũ-sãanga.
