Tãoos a yembre kibare.
Tãoos n wa n tɩ ti tãoos roogo, koom noore, n gũusd we-rũmsã, sẽn wat n yũud koomã, n tãoodẽ. La, daar a ye, a wa n kẽnga zĩigẽ wã, n na n tɩ tãoose.
La, a wa n zẽka a nenga, n yã kinkirga, sẽn zĩ n sõt a ligdi. T’a sok a menga : «M tãood-a lame n kʋ, la m sig n tɩ rɩk a ligdã bɩ? » Tɩ kinkirgã gir a zugu. : «Bɩ, m na n tɩ sõng-a lame, n sõd a ligdã, t’a yãk n kõ-ma?» Tɩ, kinkirgã gõms a zugu. Rao wã kẽngme tɩ sõnga kinkirgã, n sõd a ligdã n sa. La, kinkirgã yãka ligdã, n kõ-a, t’a tõdge, n yet t’a ka rat ye, tɩ yẽ sẽn datã, ya a bãngrã. La, kinkirgã yeela rao wã : «Fo ka tõe n zã bãngrã ye.» La, rao wã yeelame, t’a tõeeme. Kinkirgã sɩd mak n kongame, n kõ-a nug-bĩnga, la, a yeel-a, t’a ra wa saag n togs ned ye, bala, a na n kiime.
Tãoosã pag me, ra yaa nin-feoogo. Daar a ye, a pagã wa n da werda ki neerẽ, tɩ noos wa rat n dɩ ki wã. La, no-raoog a ye, n yeela no-yãanga, t’a leb pagã goabg nif sẽn ka ne wã sẽnẽ, n liib pagã, t’a peng ki wã ne wamdã, n lub tɩ b paam n dɩ. Tɩ, tãoosã zĩẽ n kelgdẽ. No-yãangã, sɩd leba pagã nug-goabgã, n liib-a t’a peng ki wã tɩ rʋʋge, tɩ b sɩd rɩ. La, pagã sɩd ra zĩime, n tar laado. La, pagã soka a sɩdã, a laadã võore, t’a sɩdã yet tɩ ka bũmb ye. (Bala, a sã n togse, a na n kiime). La, pagã yeelame, t’a sã n ka wilg-a, a laadã võore, yẽ na n wʋka a teed n kuili, a ba-yir tɩ sa.
Rao wã makame, n kong n togs-a. Dẽ poore, rao wã kiime. La, tẽngã rãmb tuu kabre, n na n mum-a. T’a zoa a ye, tõdge, n deeg kũumã, n dɩk-a, n kẽ weoogã, n bao zĩig n bĩng-a, n yaool n bao zĩig n solge, n get kũumã. La, dʋgr-bila, n wa n ta, n na n dɩ-a. A sẽn kolg n gok kũumã, la, a sẽn rok n lʋɩ, n ki. La, dʋgr naab wa n taame, n yãk tɩɩm ne zʋʋre, a korgã pʋgẽ n sɩɩs dʋgr-bilã, n yaool n yeel-a : «Fo sã n mi n yã rɩɩbo, bɩ f bas tɩ dʋgr naab goke, tɩ fo yaool n goke. Fo sã n mi n yã rɩɩbo, bɩ f bas tɩ dʋgr naab gok tɩ fo yaool n goke. Fo sã n mi n yã rɩɩbo, bɩ f bas tɩ dʋgr naab gok tɩ fo yaool n goke.» Naoor a tã, tɩ dʋgrã vʋʋge. Dʋgr naabã, kẽngame marsã, n tɩ ta, n na n dɩ kũumã. La, rao wã, yɩɩ tao-tao, n kẽng n tɩ rɩk zʋʋrã, n tɩ pĩng kũumã n yeele : «Fo sã n mi n wʋm gomde, reng n togs f zoa, la f yaool n togs f paga. Fo sã n mi n wʋm gomde, reng n togs f zoa, la f yaool n togs f paga. Fo sã n mi n wʋm gomde, reng n togs f zoa, la f yaool n togs f paga.» Naoor a tã, tɩ kũumã tĩsi, n vʋʋge.
